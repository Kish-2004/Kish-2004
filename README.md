## 🚀 Featured Project — Live on Azure

### 🚗 AI Vehicle Damage Estimator

> *Upload a photo of a damaged vehicle → AI detects the damage → instant repair cost estimate generated*

<div align="center">

[![Live Demo](https://img.shields.io/badge/🌐_Live_Demo-Azure_Cloud-0078D4?style=for-the-badge&logo=microsoft-azure&logoColor=white)](https://my-vehicle-app.eastus.cloudapp.azure.com)
[![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)](https://my-vehicle-app.eastus.cloudapp.azure.com)
[![YOLO](https://img.shields.io/badge/YOLO-Computer_Vision-FF6F00?style=for-the-badge&logo=opencv&logoColor=white)](https://my-vehicle-app.eastus.cloudapp.azure.com)

</div>

An intelligent damage assessment system powered by **YOLO (You Only Look Once)** — the industry-standard real-time object detection model. The system identifies damage regions, classifies severity, and produces a structured cost breakdown — all from a single photo.

**✨ Key Features:**

- 🧠 **YOLO Computer Vision** — real-time damage detection & region localization
- 💰 **Automated Cost Estimation** — instant repair cost breakdown by damage zone
- 📄 **PDF Report Generation** — professional downloadable damage reports
- 🔐 **Google OAuth 2.0** — secure, one-click authentication
- ☁️ **Azure Cloud Deployed** — live, production-grade deployment on Microsoft Azure
- 🐳 **Dockerized** — fully containerized for consistent deployment anywhere
